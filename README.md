# Navneet Mallick - Portfolio

A modern, responsive portfolio showcasing projects, skills, and experience in Web Development, Machine Learning, and Data Science.

## Features

- Modern dark/light theme with smooth animations
- Fully responsive design (desktop, tablet, mobile)
- Interactive project showcase with filters
- Contact form with EmailJS integration
- Performance optimized with lazy loading
- Keyboard shortcuts and accessibility features

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: CSS Grid, Flexbox, CSS Animations
- **Libraries**: Font Awesome, Google Fonts, EmailJS, Three.js
- **Hosting**: GitHub Pages

## Quick Start

1. Clone the repository
2. Open `index.html` in a browser
3. No build process required

## Features Breakdown

- **Navigation**: Smooth scroll, active link tracking, mobile hamburger menu
- **Hero Section**: Typing animation, particle effects, CTA buttons
- **Projects**: Interactive cards with hover effects, project filters
- **Skills**: Categorized tech stack with proficiency levels
- **Contact**: EmailJS form with validation and feedback
- **Theme**: Dark/light mode toggle with localStorage persistence
- **Games**: Built-in mini-games (Runner, Flappy Bird, Tetris, Minesweeper)
- **Music Player**: Background music with volume control

## Performance

- Lazy loading for images
- Throttled scroll events
- Mobile-optimized animations
- Adaptive refresh rate detection
- Low-end device detection

## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
