# Testimonials Photos

This folder contains personal photos of colleagues who provided testimonials for the portfolio.

## Privacy Notice

⚠️ **This folder is excluded from version control** via `.gitignore` to protect the privacy of individuals whose photos are stored here.

## File Naming Convention

Photos should be named using the following format:
- `FirstName.jpeg` or `FirstName.jpg`
- Example: `Prashant.jpeg`, `Aashish.jpeg`

## Image Requirements

For best display in the circular testimonial cards:
- **Recommended size**: 200x200px to 400x400px
- **Format**: JPEG or JPG
- **Aspect ratio**: 1:1 (square)
- **File size**: Keep under 200KB for optimal loading

## Adding New Testimonials

1. Add the photo to this folder with the correct naming format
2. Update `index.html` to include the new testimonial card
3. Ensure the photo filename matches the name used in the HTML

## Fallback Behavior

If a photo fails to load, the system will automatically display the person's first initial as a fallback with a gradient background.
